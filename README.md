# MasterlyApp

MasterlyApp is a modern Android application designed to help users track their learning progress and master new skills. Whether you're learning a new programming language, practicing a musical instrument, or developing professional skills, MasterlyApp provides the tools to log your hours and visualize your journey towards mastery.

## Features

- **Skill Tracking**: Create and manage a list of skills you want to master.
- **Progress Visualization**: Beautifully designed cards and progress bars to show how far you've come.
- **Time Logging**: Log hours dedicated to each skill and track your total effort against your goals.
- **Modern UI**: Built entirely with Jetpack Compose following Material 3 design principles.

## Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Dependency Injection**: [Hilt](https://developer.android.com/training/dependency-injection/hilt-android)
- **Local Database**: [Room](https://developer.android.com/training/data-storage/room)
- **Asynchronous Programming**: Coroutines and Flow
- **Architecture**: MVVM (Model-View-ViewModel)

## Getting Started

### Prerequisites

- Android Studio Ladybug | 2024.2.1 or newer
- JDK 11 or newer
- Android SDK 24+ (Minimum SDK)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kalyanbhor2312/masterly.git
   ```
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Run the app on an emulator or a physical device.
